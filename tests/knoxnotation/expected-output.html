Line merging

To merge two or more lines, use a backslash in the end of the line. This rule applies for any line type, except code blocks.

Spaces are not automatically added, so you have to use a space before the backslash to keep lines separate.

Links

Bare Links

Bare links are automatically identified, like: http://www.example.com.
Common trailing punctuation marks will not be considered part of the link, namely:

  1. Period
  2. Comma
  3. Question mark
  4. Exclamation mark
  5. Colon
  6. Semicolon
  7. Close brace

Chevron Links

For avoiding link ambiguity, use chevrons: <http://www.example.com>.

Labeled Links

  • Use curly braces before the bare link: {Example Website}http://www.example.com.
  • Or use curly braces before the chevron link: `{Example Website}`.

---

Cite admonitions are also allowed:

---Cite
Cite admonitions work similarly to blockquotes, except that the cite will display the admonition format.
